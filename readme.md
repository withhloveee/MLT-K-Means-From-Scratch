# K-Means From Scratch

> Acknowledgement: The theory, concepts, and algorithm used in this project were learned from the Machine Learning Techniques (MLT) course offered as part of the IIT Madras BS Degree in Data Science and Applications program. This repository is a personal from-scratch implementation created for learning and educational purposes.

A simple implementation of the K-Means clustering algorithm using NumPy and Python, built from scratch for learning and educational purposes.


# Algorithm:
- assume, we already know the K value.
- step1: initialize the z(0)
- step2: find mean using the z(0)
- step3: assign the new z using the new mean call it z(1)
- repeat until the z(t) = z(t+1)
